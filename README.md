# UNO

## How to run:

### Install dependencies:
 - ```cmd
   pip install asyncio
   ```
 - ```cmd
   pip install quart
   ```
- ```cmd
   pip install websockets
   ```
- ```cmd
  pip install qrcode
  ```

### Run the scripts:
```cmd
python main.py
```

and on different console:
```cmd
python start_site.py
```

### Scan QR code in console or copy URL and paste in browser
